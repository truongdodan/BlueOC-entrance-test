# React-Redux Posts Application

A simple React application that fetches and displays posts from JSONPlaceholder API using Redux for state management.

## Features

- Fetch and display posts from API
- Add new posts (simulated)
- Redux Toolkit for state management and API calls
- Clean and responsive UI

## Technologies Used

- React
- Redux Toolkit
- React Redux
- JavaScript (ES6)
- JSONPlaceholder API
- CSS

## Installation

1. Clone the repository:

```bash
git clone git@github.com:truongdodan/BlueOC-entrance-test.git
cd Task3_React-ReduxApplication
```

2. Install dependencies:
   npm install

3. Run the development server:
   npm run dev

4. Open your browser and visit:
   http://localhost:5173
